HTML versus XHTML

XHTML é uma versão de HTML mais estrita e mais baseada em XML.

O que é XHTML?

XHTML significa E X tensible H yper T ext Markup Language

XHTML é uma versão de HTML mais estrita e mais baseada em XML

XHTML é HTML definido como um aplicativo XML

XHTML é suportado por todos os principais navegadores

Por que XHTML?

XML é uma linguagem de marcação onde todos os documentos devem ser marcados corretamente (ser "bem formados").

XHTML foi desenvolvido para tornar o HTML mais extensível e flexível para trabalhar com outros formatos de dados (como XML). Além disso, os navegadores ignoram os erros nas páginas HTML e tentam exibir o site mesmo que ele tenha alguns erros na marcação. Portanto, o XHTML vem com um tratamento de erros muito mais rígido.
